OSI Model

The OSI model contains 7 layers:

1. Application Layer
2. Presentation Layer
3. Session Layer
4. Transport Layer
5. Network Layer
6. Data Link Layer
7. Physical Layer

TCP/IP Model

The TCP/IP model contains 4 layers:

1. Application Layer
2. Transport Layer
3. Internet Layer
4. Network Access Layer
